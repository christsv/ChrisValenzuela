2017: Sports Sensor Arm Sleeve (SASS)


This project was created to improve a basketball player's shot "form." The user wears a sport sleeve that instantly records the angle of their release with respect to the ground. It operates with two Arduino Nano microcontrollers that recieve and process data from two accelerometers that are attached to the sleeve. After several formulas, the calculated angle is displayed onto an LCD screen also attached to the sleeve! The link below goes more in depth with this project. Check it out!
